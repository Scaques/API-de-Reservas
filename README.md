# API-de-Reservas
API RESTful empresarial para la gestión y validación de reservas en tiempo real. Desarrollada con Java 17 y Spring Boot 3.3.4 bajo una arquitectura limpia en capas. Cuenta con un escudo de seguridad con Jakarta Bean Validation para asegurar la integridad de los datos y persistencia automatizada mediante Spring Data JPA y base de datos H2.
